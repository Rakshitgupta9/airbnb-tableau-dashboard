# 🏡 Airbnb Listings Data Analysis Dashboard (Tableau)

## 📊 Project Overview

This project focuses on analyzing the Airbnb Listings 2016 dataset to uncover insights related to pricing, revenue trends, and geographic distribution of listings.

The dashboard was built using Tableau by combining listing data and calendar data to perform end-to-end analysis and create an interactive visualization.

---

## 🎯 Business Questions Solved

* How do prices vary across different zip codes?
* What are the weekly revenue trends?
* How does the number of bedrooms impact pricing?
* Which locations have higher average listing prices?

---

## 🛠 Tools & Technologies Used

* Tableau (Data Visualization & Dashboarding)

* Data Cleaning & Preparation
