# 🏡 Airbnb Listings Data Analysis Dashboard (Tableau)

## 📊 Project Overview

This project analyzes the Airbnb Listings 2016 dataset to uncover insights related to pricing, revenue trends, and geographic distribution of listings.

Using Tableau, an interactive dashboard was developed to explore how factors such as location, number of bedrooms, and time impact Airbnb pricing and revenue patterns.

---

## 🎯 Business Questions Solved

* How do Airbnb prices vary across different zip codes?
* What are the weekly revenue trends over time?
* How does the number of bedrooms affect listing prices?
* Which regions generate higher revenue and pricing?

---

## 🛠 Tools & Technologies Used

* Tableau (Data Visualization)
* Data Analysis & Exploration
* Kaggle Dataset (Airbnb Listings 2016)

---

## 🔄 Data Analysis Workflow

### ✅ Data Preparation

* Used Airbnb listings and calendar datasets
* Structured data for analysis in Tableau
* Combined listing and pricing information

### ✅ Data Visualization

* Built multiple charts to analyze trends and patterns
* Used geographic mapping for location-based insights
* Designed interactive filters for better exploration

---

## 📈 Dashboard Features

* 📍 **Geographical Map** → Price distribution by zipcode
* 📉 **Weekly Revenue Trend** → Time-based revenue analysis
* 📊 **Average Price by Zipcode** → Comparison across regions
* 🛏 **Price vs Bedrooms** → Relationship between size and pricing
* 🎛 **Interactive Dashboard** → Dynamic filtering and exploration

## 📌 Key Insights

* Prices vary significantly across zipcodes, indicating location-based demand
* Revenue shows steady trends with slight seasonal variations
* Listings with more bedrooms tend to have higher average prices
* Certain zipcodes dominate high-value listings
